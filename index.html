Sou estudante de Engenharia de Software e desenvolvedor com foco em Java e Spring Boot, especializado no desenvolvimento de APIs RESTful e na integração com bancos de dados relacionais, como PostgreSQL e Oracle DB. Busco estar aprimorando minhas habilidades e buscando aplicar novas tecnologias para desenvolver soluções eficientes, escaláveis e seguras.
